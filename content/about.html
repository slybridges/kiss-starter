You can also write content in HTML

Might be convenient for pages with complex design that you cannot write in Mardown directly

This page doesn't have any image. In that case, Kiss uses the default image as per the config when displayed in collections and shared online.

You can also take a look at the default optimizations done by kiss:

  • SEO and sharing: checkout the default.njk template to see how kiss can easily add meta and Open Graph tags.
  • Images: checkout the source code and public folder to see how kiss generates responsive images.
  • RSS: the feed.xml file in the public folder is the RSS (Atom) feed generated by kiss.
  • Sitemap: the sitemap.xml file in the public folder is the sitemap you can give to Google or Bing to help them crawl your site.
  • Sitedata: the sitedata.json file in the public folder is the JSON dump of your site data. Use this data for debug.
